# 基于Vue.js的一款音乐播放器
-------

### 项目介绍
项目是基于Vue.js的移动端的音乐播放器，总体是vue-cli + webpack实现了整体框架的搭建，数据是基于QQ音乐，做到了跨域请求

#### 视图
* 推荐页
* 歌手页
  * 歌手详情
* 排行
* 搜索
* 歌曲播放

#### src目录结构
> api → 数据抓取
> base → 公共基础组件
> common → 公共代码
  >> fonts
  >> image
  >> js
  >> stylus 
> components → 公共组件
> router → 项目路由
> store → 状态管理
